About Me

Currently, I am a student in the area of Computer Engineering at ISEP in Porto, Portugal. In this year of 2019 in parallel with the university course I'll start to take my CCNA Cisco certification and I'm also learning IOS development (with swift)

I always knew that this was the area I wanted to work with. Since then I have gone into various techniques and languages.


My Courses

In 2018 I started my university course that is Computer Engineering (in portuguese: Engenharia Informática) where I've learned Java Principles, Principles of computing and work in a group project (about how to analize social networks). Now I'm learning about Software Engineering and Java OOP.

In 2019 I started my trail to obtain the Cisco CCNA certification.

Outside university I started doing a modern web course where I've learned an intermediate javascript base until the current phase of lessons (progress: 142/498 itens).

Already, started the course iOS 12 & Swift - The Complete iOS App Development Bootcamp (progress: 170/527 itens), where I've learned until now: read and using apple documentation, MVC Design Pattern, iOS auto layout and constraints, Swift OOP, how to use cocoapads, request api data.
